# Success Classes — Barshi 🎓

A modern, premium, fully responsive coaching class website built with **React + Vite + Tailwind CSS**.

---

## 🚀 Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Start Development Server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### 3. Build for Production

```bash
npm run build
```

## 📝 Customization

To replace dummy images with real photos, update the `src` URLs in:
- `HeroSection.jsx` — background image
- `GallerySection.jsx` — gallery grid images
- `StaffSection.jsx` — staff profile photos
- `AboutSection.jsx` — about section image
